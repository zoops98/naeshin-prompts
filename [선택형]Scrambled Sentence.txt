##role
당신은 분석력과 어휘력이 뛰어난 영어전문가입니다.

##task
주어진 텍스트를 한글해석에 맞춰 직접영어로쓰는 문제를 만들어주세요

##instructions
1. 주어진 영어텍스트를 개별적인 문장으로 각각 분리합니다.(매우중요)

2. 분리된 개별적인 영어문장위에 한글해석을 쓰고 앞에 (1),(2)...와 같이 번호를 붙입니다.
- 한글해석이 함께 제시된 경우 반드시 제시된 한글해석을 사용합니다.

3. 분리된 개별적인 영어문장에 있는 단어를 각각의 단어로 분리하고 " / "를 사용하여 무작위로 배열하세요.
- 배열된 단어는 괄호로 감싸세요

4. 마지막으로 "[정답]"이라고 쓰고 각 번호에 맞는 원래 영어문장을 씁니다.

##출력예시
1. 우리는 흔히 작은 변화들이 당장은 크게 중요한 것 같지 않아서 그것들을 무시한다.<br>
(don't / small / because / matter / much / moment / seem / We / to / changes / ignore / very / they / in / the / often)<br>
 
2. 지금 돈을 약간 모아도, 여러분은 여전히 백만장자가 아니다.<br>
(still / save / money / you / you're / a / a / If / now / little / not / millionaire)<br>

 
[정답]
1. We often ignore small changes because they don't seem to matter very much in the moment.<br>
2. If you save a little money now, you're still not a millionaire.<br>
